# Araon v1.1.0-beta.10

Status: beta release, `CONDITIONAL GO`

Araon v1.1.0-beta.10 publishes the P1/P2 product-completion work that landed
after v1.1.0-beta.9. This release keeps the beta channel moving without
changing `npm latest` or promoting a stable release.

## Highlights

- Persisted realtime signal timeline for deterministic watch reasons.
- Observation notes/log MVP in the stock detail flow.
- News and disclosure external-link panels with local cache support.
- ETF/ETN/REIT/fund grouping improvements and a false REIT classification fix.
- Pinned candle inspection in the StockDetailModal chart tab.
- Candle chart crosshair tooltip with KST time, OHLCV, volume, and source.
- Selected-ticker today-minute backfill route and single-ticker live write
  verification for `005930`.
- KIS daily backfill single-ticker verification remains part of the chart
  baseline.
- Data retention and data-growth diagnostics for signals, notes, news, candles,
  and maintenance state.
- Data health panel expansion for operational visibility.
- Volume surge baseline bootstrap from local snapshot history.
- P1/P2 product verification and integration acceptance documentation.

## Runtime And Data Policy

- Araon remains a localhost personal watchlist tool.
- Fresh installs have no credentials and make no external KIS calls.
- After credentials are configured, managed cap40 realtime and guarded daily
  backfill are enabled by default.
- REST polling remains the fallback path.
- Selected minute backfill is manual and selected-ticker only.
- Full watchlist minute backfill remains prohibited.
- Background or automatic historical minute backfill remains prohibited.
- Missing chart data is not synthesized.
- Raw tick permanent storage remains out of scope.

## Security

- Araon has no trading or order-entry feature.
- Do not commit `credentials.enc`, local data directories, SQLite databases,
  runtime logs, tokens, approval keys, or account identifiers.
- The package contents audit must exclude local runtime state and secrets.

## Known Limitations

- Release status is `CONDITIONAL GO`, not stable/latest.
- `npm latest` is not updated.
- Full watchlist minute backfill remains HOLD.
- Automatic/background historical minute backfill remains HOLD.
- News remains external-link/cache based; Araon does not provide news analysis,
  summarization, or sentiment scoring.
- Long-run production observation is still recommended for the new retention
  and data-growth paths.
- DMG/EXE GUI manual validation should be repeated for this beta if desktop
  assets are published later.
- Desktop builds remain unsigned.
