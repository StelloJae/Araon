# Araon v1.1.2

Status: stable desktop patch release for the localhost Araon monitor.

Araon v1.1.2 fixes the Windows installer package that was published with
v1.1.1. The previous Windows desktop artifact was built from an Apple Silicon
machine without forcing the Windows architecture, so the generated installer
contained an ARM64 Windows payload while the public download did not make that
clear. On a typical x64 Windows machine this could install the uninstaller
without a usable Araon desktop executable.

## Fix

- Windows desktop builds now target `x64` explicitly.
- Windows release assets now include the architecture in the filename:
  - `Araon-Setup-1.1.2-x64.exe`
  - `Araon-1.1.2-x64-portable.exe`
- The NSIS setup and portable artifact names are now explicit and GitHub-safe.
- A regression test now locks the Windows desktop build configuration so future
  macOS-hosted builds do not silently produce ARM64-only Windows installers.

## Known Limitations

- Windows execution still needs final manual validation on a real Windows
  machine after downloading the v1.1.2 installer.
- Desktop artifacts are unsigned and may trigger Windows SmartScreen or macOS
  Gatekeeper warnings.
- Araon remains a read-only localhost monitoring tool with no order-entry
  feature.
