# Araon v1.1.0-beta.4

Araon v1.1.0-beta.4 combines the desktop beta artifacts, CLI launcher beta,
and the latest dashboard polish from main.

## Download

Desktop beta artifacts are unsigned:

- macOS: `Araon-1.1.0-beta.4-arm64.dmg`
- macOS zip: `Araon-1.1.0-beta.4-arm64-mac.zip`
- Windows installer: `Araon.Setup.1.1.0-beta.4.exe`
- Windows portable: `Araon.1.1.0-beta.4.exe`

macOS Gatekeeper and Windows SmartScreen may warn because these builds are not
signed or notarized yet.

## CLI launcher

Run the npm beta directly:

```bash
npx @stellojae/araon@beta
```

During the beta period, use the explicit `@beta` tag.

Or install the npm beta globally:

```bash
npm install -g @stellojae/araon@beta
araon
```

The CLI starts Araon on `127.0.0.1`, serves the built React frontend, and opens
the default browser. Use `Ctrl+C` to stop it gracefully.

## Dashboard polish

- Master-search stock adds now re-sync tracked stocks from `/stocks`, so
  server-derived `autoSector` appears without a manual refresh.
- Sector row flash now indicates price-relevant movement only: price,
  percentage change, or absolute change.
- Timestamp-only and volume-only updates no longer flash sector rows.
- Compact sector rows no longer show the volume pill, avoiding overlap with
  price and change columns.
- Integrated realtime copy now shows the verified status instead of
  "experimental"; cap controls are tucked under an operator recheck section.

## Runtime policy

- Fresh installs keep realtime disabled by default.
- REST polling remains the fallback path.
- KIS credentials are required before live market data can start.
- Araon is localhost-only and read-only.
- Araon does not place orders or execute trades.
- Docker is not included in this beta.
- Apple notarization and Windows code signing are not included in this beta.
