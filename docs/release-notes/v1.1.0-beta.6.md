# Araon v1.1.0-beta.6

Araon v1.1.0-beta.6 is a dashboard correctness prerelease for realtime surge
visibility and official KIS industry grouping.

## Install

During the beta period, use the explicit npm beta tag:

```bash
npx @stellojae/araon@beta
```

Or install globally:

```bash
npm install -g @stellojae/araon@beta
araon
```

GitHub Release assets may also include a packaged npm tarball and unsigned
desktop beta artifacts.

## Fixes

- Realtime 10-30 second surge signals now allow the first usable baseline to
  create a signal when the move is already above threshold.
- `/stocks` now derives display grouping from KIS official index industry codes.
- KRX sector flags are no longer used as a display grouping fallback.
- Sector grouping is now: manual sector > KIS official index industry >
  unclassified.
- Stocks without official KIS index industry, including ETF/ETN/special
  products, remain in `미분류`.
- Sector row pills, section headers, and detail modal labels now use the same
  source wording.

## Runtime policy

- Fresh installs keep realtime disabled by default.
- REST polling remains the fallback path.
- KIS credentials are required before live market data can start.
- Araon is localhost-only and read-only.
- Araon does not place orders or execute trades.
- Docker is not included in this beta.
- Apple notarization and Windows code signing are not included in this beta.
