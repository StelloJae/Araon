# Araon v1.1.1

Status: stable patch release for the localhost Araon monitor.

Araon v1.1.1 fixes a premarket quote routing issue in the REST polling lane.
The app was already receiving NXT premarket ticks for WebSocket-covered
tickers, but the REST fallback lane still queried KRX-only current prices.
That made non-favorite tracked stocks look static during the 08:00-09:00
premarket window.

## Fix

- REST polling now uses the KIS current-price market code that matches the
  session:
  - `NX` during NXT premarket and after-hours windows.
  - `UN` during the regular integrated session and closed windows.
- The NXT WebSocket route remains unchanged:
  - NXT feed during premarket/after-hours.
  - Integrated feed during the regular session.

## Known Limitations

- REST polling still depends on KIS current-price endpoint availability and
  market-session behavior.
- WebSocket realtime remains capped at the KIS subscription ceiling.
- Full watchlist minute backfill and automatic historical minute backfill remain
  prohibited.
- Araon remains a read-only localhost monitoring tool with no order-entry
  feature.
