# Araon v1.1.0-beta.2

Araon v1.1.0-beta.2 is the unsigned desktop beta channel for the Electron-packaged Araon app.

## Scope

- macOS unsigned desktop artifact
- Windows unsigned desktop artifact
- Electron shell around the existing localhost-first Fastify and React app
- Runtime data stored under the OS user-data directory
- Fresh install realtime defaults remain OFF

## Not Included

- Apple notarization
- Windows code signing
- Docker packaging
- Changes to KIS live/WebSocket/cap validation policy
- Any change to the existing v1.0.0 release

## Operator Notes

Unsigned desktop artifacts can show OS warnings:

- macOS: Gatekeeper may block or warn until the app is explicitly allowed.
- Windows: SmartScreen may warn because the binary is unsigned.

Araon remains a read-only monitoring tool. It does not place orders or execute
trades.
