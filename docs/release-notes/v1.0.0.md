# Araon v1.0.0

Araon v1.0.0 is the first public open-source release of the localhost-first
Korean stock watchlist dashboard.

## Highlights

- Korean stock watchlist dashboard for KIS OpenAPI users
- KIS REST polling fallback for tracked stocks
- KIS `H0UNCNT0` integrated WebSocket realtime feed
- cap40 controlled realtime acceptance completed before release
- Session-scoped realtime controls and rollback from the local UI
- KIS master catalog for searchable KOSPI/KOSDAQ universe data
- KIS-derived `autoSector` classification with manual override support
- Korean stock-name 초성 search
- Server-Sent Events for live UI updates
- Araon favicon and Apache-2.0 project metadata
- Trustworthy volume-surge baseline foundation

## Requirements

- Node.js 20 or newer
- npm
- KIS OpenAPI app key/app secret
- Local browser access to `http://127.0.0.1:5173`

## Security Notes

- Araon is a read-only local monitoring tool.
- It does not place orders or execute trades.
- KIS credentials are entered through the local UI and encrypted under `data/`.
- Do not commit `data/`, `.env`, credentials, account identifiers, access
  tokens, or approval keys.

## Known Limitations

- Fresh installs keep realtime OFF until the user enables it.
- Volume-surge ratios appear only after enough baseline samples exist.
- Docker Compose packaging is planned after v1.0.0.
- Windows service / task scheduler packaging is planned after v1.0.0.
- Screenshots are not bundled in this release.
