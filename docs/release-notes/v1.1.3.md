# Araon v1.1.3

Status: stable patch release for the localhost Araon monitor.

Araon v1.1.3 fixes a live dashboard issue where some favorites could appear as
`0` or stop changing even while the real market price was moving.

## Fix

- Realtime-tier tickers now keep REST fallback polling until Araon has received
  a valid integrated WebSocket quote for that ticker.
- REST polling no longer writes non-positive quotes into the live price store.
  This prevents a temporary `0` quote from replacing a valid displayed price.
- Regression tests cover both the realtime fallback gate and non-positive REST
  quote handling.

## Known Limitations

- Araon still depends on KIS data availability by market phase and route.
- Desktop artifacts remain unsigned and may trigger Windows SmartScreen or
  macOS Gatekeeper warnings.
- Araon remains a read-only localhost monitoring tool with no order-entry
  feature.
