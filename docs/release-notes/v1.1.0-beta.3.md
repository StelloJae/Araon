# Araon v1.1.0-beta.3

Araon v1.1.0-beta.3 is the planned CLI launcher beta channel.

## CLI launcher

- Adds the `araon` binary entry for running Araon from a terminal.
- Starts the local Fastify server on `127.0.0.1`.
- Serves the production React frontend from the built client bundle.
- Opens the default browser unless `--no-open` is used.
- Supports `--port`, `--host`, `--data-dir`, `--log-level`, `--version`, and `--help`.
- Supports optional `--exit-when-browser-closes` shutdown via browser heartbeat.

## Data policy

- CLI data directory priority is `--data-dir`, then `ARAON_DATA_DIR`, then the OS user-data default.
- The CLI does not default to writing runtime state into the repository `data/` directory.
- Fresh installs keep realtime disabled by default.

## Limits

- This beta is still localhost-only and read-only.
- It does not add order placement or trading.
- It does not change the unsigned Electron beta signing/notarization status.
