# Araon v1.1.0-beta.3

Araon v1.1.0-beta.3 is the CLI launcher beta channel.

## CLI launcher

- Adds the `araon` binary entry for running Araon from a terminal.
- Starts the local Fastify server on `127.0.0.1`.
- Serves the production React frontend from the built client bundle.
- Opens the default browser unless `--no-open` is used.
- Supports `--port`, `--host`, `--data-dir`, `--log-level`, `--version`, and `--help`.
- Supports optional `--exit-when-browser-closes` shutdown via browser heartbeat.

## Install and run

Install from the release npm package tarball:

```bash
npm install -g ./araon-1.1.0-beta.3.tgz
araon
```

Or run from the source archive:

```bash
npm install
npm run build
npm run start:cli
```

Node.js 20 or newer is required.

## Data policy

- CLI data directory priority is `--data-dir`, then `ARAON_DATA_DIR`, then the OS user-data default.
- The CLI does not default to writing runtime state into the repository `data/` directory.
- Fresh installs keep realtime disabled by default.
- KIS credentials are required before live market data can start.
- Live realtime remains user-controlled through settings/session controls.

## Limits

- This beta is still localhost-only and read-only.
- It does not add order placement or trading.
- Electron is not included in this CLI beta.
- Docker is not included in this CLI beta.
- The unsigned desktop app remains a separate beta channel.
- Native dependency installation may require local build tools depending on OS.
- Volume surge ratios need baseline samples before ratio display becomes available.
